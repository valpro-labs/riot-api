# Riot API (unofficial)

[![npm](https://img.shields.io/npm/v/@valpro-labs/riot-api.svg)](https://www.npmjs.com/package/@valpro-labs/riot-api)

A typed TypeScript client for Riot Api. This library provides easy access to riot endpoints with full TypeScript support.

## Installation

```bash
npm install @valpro-labs/riot-api
```
